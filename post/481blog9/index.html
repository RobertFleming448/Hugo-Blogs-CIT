Project Description

For this project we needed to create a typing speed test app. The project requirements were as follows:

  • A timer to measure the test
  • A text-entry box to check if a user types the phrase correctly
  • A notification for typing progress
  • A top score table to register high scores

Timer

I tried several methods to create a timer, but the one that ended up working was Date.now() function. If you run the Date.now() function you will get the current time in standard format. By taking the difference between two occurrences of this command you can get the range that has occurred in milliseconds. This can be further processed to get the standard minute:second:millisecond format. A mistake I made at first was to run this function too often. This caused the browser to freeze, I changed the interval and the timer worked while accepting other user input at the same time code for the timer is below:

function startTimer() {
  if (running === true) {
    checkText();
  }
  else if (finished === true) {
    return;
  }
  else {
    running = true;
    start = Date.now();
    interval = window.setInterval(stopWatch, 1);
  }
}
function stopWatch() {
	total = Date.now() - start;
	min = Math.floor(total / 60000);
	sec = Math.floor((total % 60000) / 1000);
	mil = Math.floor(((total % 60000) % 1000) / 10);
	setValue();
}

function format(total) {
	min = padZero(Math.floor(total / 60000));
	sec = padZero(Math.floor((total % 60000) / 1000));
	mil = padZero(Math.floor(((total % 60000) % 1000) / 10));
	return min + ":" + sec + ":" + mil;
}

Text Box

Event Listeners

The text box required event listeners to track user progress. Event listeners as the name implies are tools in JavaScript that can detect certain page conditions to trigger certain events. I needed to listen for the user starting the test and the user resetting the test. For this I used the keyup and click events respectively. The keyup will detect when the user finishes their first keystroke, and the click will simply listen for left clicks.

testArea.addEventListener('keyup', startTimer);
resetButton.addEventListener("click", resetAll);

Error Notification

Next I needed to check the user input to see if the answer was correct. To check it all I needed to do was get the fields contents, and change some CSS based on the value of those contents. I managed to do that with the function below:

function checkText() {
  let response = document.getElementById("test-area").value;
  let sample = document.getElementById("sample").innerHTML;
  if (sample === response) {
    document.getElementById("test-wrapper").style.border = "10px solid yellow";
    window.clearInterval(interval);
    document.getElementById("words").innerHTML = "Words per Minute: " + Math.floor(540000 / total);
    running = false;
    finished = true;
    scoreCheck();
  }
  else if (sample.substring(0, response.length) === response) {
    document.getElementById("test-wrapper").style.border = "10px solid grey";
  }
  else {
    document.getElementById("test-wrapper").style.border = "10px solid red";
    errors++;
    document.getElementById("errors").innerHTML = "Errors: " + errors;
  }
}

Top Score

To create the high score system I decided it would be best to store the values in an array:

const bigScore = 6061000;
let topScores = [bigScore, bigScore, bigScore, bigScore, bigScore, bigScore, bigScore, bigScore, bigScore, bigScore];

This constant represents the highest possible score. Therefor any new score will register as a high score. To add a new score I push a new value into the array sort it and finally pop the worst time.

The Costly Mistake

I was certain I wanted to use a sort function to get the order. I thought this would simplify the process but the opposite was true. What I didn’t know about the standard sort method, is that is sorts alphabetically. This may seem obvious to someone well versed in JavaScript but to me a relative novice it was very foreign. In hindsight this makes a lot of sense, since JavaScript variables are untyped. If I were to choose a default sorting method for an array I would have picked alphabetically as well. With that adjusted and many hours of troubleshooting it worked as intended. The code for sorting and setting the table elements is listed below:

function scoreCheck() {
	topScores.push(parseInt(total));
	topScores.sort(function (a, b)
		{
			return a - b;

	topScores.pop();
